# flowtine
a json-style parser library made for C++, used for making simple declerations

## Features

### Declaring variables
To declare variables, you can do: <br />
<code>
  var name = value; 
</code>

To access the value in C++, do: <br/>
<code>
  ftn::val("name"); <br />
  // Output: value
</code>
